jglCreateTexture

Function for generating jgl textures. This function does different things depending on what it is given. If a 1D array is passed in, the array is replicated down to make a square and the resulting texture is returned, the texture is using grayscale. If a 2D array is passed, a greyscale texture is created and returned. If a 3D array is passed, if it is NxMx3 an RGB texture is returned, and if it is NxMx4 and RGB and Alpha texture is returned.

jglCreateTexture(array: Array): any
Parameters
array (Array) the array to pass in.
Returns
any: the texture

jglBltTexture

Function for drawing the given texture to screen. All params except texture are optional, defaults to center with 0 rotation.

jglBltTexture(texture: Object, xpos: Number, ypos: Number, rotation: Number)
Parameters
texture (Object) the texture to draw, should only pass something given by jglCreateTexture.
xpos (Number) the x-coordinate to place the center of the texture.
ypos (Number) the y-coordinate to place the center of the texture.
rotation (Number) the rotation of the texture in degrees.
